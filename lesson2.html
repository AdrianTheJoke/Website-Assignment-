Lesson 2: HTML Basics

In this lesson, you will learn the fundamentals of HTML, the markup language that forms the structure of a webpage.

What is HTML?

HTML (Hypertext Markup Language) is the standard language used to create webpages. It structures the content on the web, such as text, images, and links.

HTML Tags

HTML uses "tags" to define elements. Some common tags include:

  • <h1> to <h6>: Defines headings (from largest to smallest).
  • <p>: Defines a paragraph of text.
  • <a>: Defines a hyperlink.
  • <img>: Defines an image.
  • <ul> and <li>: Defines an unordered list and its list items.

Why Learn HTML?

HTML is essential for web development, as it forms the backbone of every website. It allows you to organize content, add multimedia, and create links to other pages.
