CSS Selectors and Properties

1. Color

CSS uses the color property to change the text color. For example:

This text is red.

color: red;

2. Background Color

The background-color property sets the background color of an element. For example:

This box has a blue background.

background-color: blue;

3. Font Size

The font-size property controls the size of the text. For example:

This text is 16px in size.

font-size: 16px;

4. Margin

The margin property adds space around an element. For example:

This box has 20px of margin around it.

margin: 20px;

5. Padding

The padding property adds space inside an element. For example:

This box has 10px of padding inside.

padding: 10px;
